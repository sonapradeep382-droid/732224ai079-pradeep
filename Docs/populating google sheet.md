- Sign in into N8N.
- Open n8n and click New Workflow.
- Add a Click Trigger node as the trigger.
- Add an Get row from sheets.
- Connect Click trigger node to Get row from Sheets node.
- Add an If condition node.
- Add a condition in the If node Settings.
- Connect True branch with a append sheets node and connect false branch with another node
- And fill them with separate sheets.
- Now Run the Workflow.
